# Business Document Templates

Clean, responsive and print-ready HTML templates for invoices, quotations, delivery notes and receipts.

## Included
- Invoice
- Quotation
- Delivery note
- Receipt

## Features
- A4 print optimization
- Responsive design
- Print / Save as PDF
- Shared CSS design system
- No framework or build step
- MIT licensed

## Run
Open `index.html` in a browser or publish the repository root with GitHub Pages.
